Introduction to Java Programming ‚òï

Welcome to the world of Java - one of the most popular and versatile programming languages in the world! Java powers billions of devices, from smartphones to enterprise servers, making it an essential skill for any aspiring programmer.

What is Java? ü§î‚Äã

Java is a high-level, object-oriented programming language that was designed to have as few implementation dependencies as possible. It's known for its "write once, run anywhere" philosophy, meaning Java code can run on any platform that supports Java without needing to be recompiled.

Key Features of Java üåü‚Äã

1. Platform Independence üåç‚Äã

  • Write Once, Run Anywhere (WORA)
  • Java code compiles to bytecode that runs on any Java Virtual Machine (JVM)
  • Perfect for cross-platform development

2. Object-Oriented Programming üèóÔ∏è‚Äã

  • Everything in Java is an object
  • Supports encapsulation, inheritance, and polymorphism
  • Promotes code reusability and maintainability

3. Memory Management üß†‚Äã

  • Automatic garbage collection
  • No manual memory management required
  • Prevents memory leaks and crashes

4. Security üîí‚Äã

  • Built-in security features
  • Sandbox execution environment
  • No explicit pointer operations

Object-Oriented Programming in Java üéØ‚Äã

Class and Object Example‚Äã

public class Car {
    // Instance variables
    private String brand;
    private String model;
    private int year;
    
    // Constructor
    public Car(String brand, String model, int year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
    }
    
    // Methods
    public void startEngine() {
        System.out.println(brand + " " + model + " engine started!");
    }
    
    public String getCarInfo() {
        return year + " " + brand + " " + model;
    }
    
    // Getters and Setters
    public String getBrand() {
        return brand;
    }
    
    public void setBrand(String brand) {
        this.brand = brand;
    }
}

// Using the class
public class Main {
    public static void main(String[] args) {
        Car myCar = new Car("Toyota", "Camry", 2023);
        myCar.startEngine();
        System.out.println(myCar.getCarInfo());
    }
}
